FAQ

常见问题

这和官方 Hermes Agent 是什么关系？

Hermes Agent 是开源基础能力；Hermes Cloud 是基于它做的商业化托管与产品化交付层。

我还需要自己维护服务器吗？

不需要。部署、守护、备份、更新和基础监控由平台负责。

支持哪些消息渠道？

可从 Telegram、Discord、WhatsApp、Email 等主流渠道起步，后续按产品路线扩展。

适合哪些用户？

适合个人生产力用户、创始人、内容团队、增长团队，以及需要长期运行 Agent 的组织。
